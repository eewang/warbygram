== Warby Parker + Instagram

Project for [Warby Parker](http://www.warbyparker.com) using Instagram API.
Built primarily on ActiveAdmin.


== Rake Tasks to run

1. lib/tasks/import_product_csv.rake:
       rake import:glasses
       rake import:color

2. lib/tasks/warbyparker.rake
       rake warby:tags
       rake warby:all_photos
       rake warby:some_photos
       rake warby:get_location_photos
